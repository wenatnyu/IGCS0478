补码计算方法 / Two's Complement Methods

方法一: 二进制值转补码Method 1: Binary to Two's Complement

步骤 1: 写出二进制值并找到从右起第一个"1"。Write binary value and locate the first "1" from the right.

步骤 2: 切换该"1"左侧的所有位 (不切换该"1")。Switch all bits to the left of that "1" (don't switch the "1" itself).

示例 / Example: 1101100 → Two's Complement

原始 / Original: 1101100

结果 / Result: 0010100

方法二: 负值转补码Method 2: Negative Value to Two's Complement

步骤 1: 找到数值的二进制等价物 (忽略负号)。Find binary equivalent ignoring the minus sign.

步骤 2: 将二进制值转换为补码 (使用方法一)。Convert binary value to two's complement (use Method 1).

步骤 3: 确保 MSB 为 1。Make MSB 1, if not already.

示例 / Example: -10 → Two's Complement

10₁₀ = 00001010₂

00001010

结果 / Result: 11110110₂

快速识别技巧与常见错误Quick Recognition Tips & Common Mistakes

  • MSB = 0: 一定是正数 / Always positive
  • MSB = 1: 一定是负数 / Always negative
  • 10000000: 最小的负数 (-128) / Smallest negative
  • 11111111: -1 (所有位都是1) / -1 (all bits are 1)
