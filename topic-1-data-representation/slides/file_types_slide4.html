MIDI格式介绍 /
MIDI Format Introduction

MIDI全称

Musical Instrument Digital Interface

音乐乐器数字接口

主要用途 / PRIMARY USES

  • 音乐文件存储 / Storage of music files
  • 电子乐器间通信 / Communication between electronic instruments
  • 数字音乐制作 / Digital music production

核心特点 / KEY FEATURES

  • 1
    指令存储方式

    存储为一系列指令而非实际音乐音符 / Stored as a series of instructions, not actual music notes

  • 2
    8位串行传输

    使用8位串行传输（异步） / Uses 8-bit serial transmission (asynchronous)

  • 3
    通信协议

    允许电子乐器互相交互的通信协议 / Protocol allowing electronic instruments to interact

优势 / ADVANTAGES

  • 文件尺寸小 / Small file size
  • 可编辑性强 / Highly editable
  • 跨平台兼容 / Cross-platform compatibility
