ASCII字符集 / ASCII Character Set

特点说明 / Features

  • 标准ASCII码字符集由7位代码组成 / Standard ASCII character set consists of 7-bit code
  • 代表标准键盘上的字母、数字和字符 / Represents letters, numbers and characters on standard keyboard
  • 包含32个控制码 / Includes 32 control codes
  • 大小写字符具有不同ASCII值 / Uppercase and lowercase have different ASCII values
  • ASCII使用一个字节来存储值 / ASCII uses one byte to store value

重要ASCII值 (十进制) / Important ASCII Values (Decimal)

字符 / Character 描述 / Description 十进制值 / Decimal Value
'0' 数字零 / Digit Zero 48
'A' 大写字母A / Uppercase A 65
'a' 小写字母a / Lowercase a 97
计算规则 / Calculation Rule: 每个后续值是前一个值+1 / Every subsequent value is previous value + 1
e.g. 'b' = 98 (97+1), 'c' = 99 (98+1)
特殊性质 / Special Property: 大小写转换：第6位从1变为0 / Case conversion: 6th bit changes from 1 to 0
