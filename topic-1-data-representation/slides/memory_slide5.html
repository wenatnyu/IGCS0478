传统系统 / Conventional System (Decimal-based)

系统特点 / System Characteristics

  • 也称为十进制系统 / Also called Decimal System
  • 使用 10ⁿ 作为基础 / Uses 10ⁿ as base
  • 每个单位 = 1000 × 前一个单位 / Each unit = 1000 × previous unit
  • 日常生活中常用的系统 / Commonly used system in daily life

关键特点 / Key Features

  • 使用传统名称 (kilo, mega, giga) / Uses traditional names
  • 与国际单位制(SI)一致 / Consistent with SI units
  • 在网络和营销中广泛使用 / Used in networking and marketing
  • 简单易于理解和记忆 / Simple to understand and remember

常见应用 / Common Applications

  • 网络带宽测量 (Mbps, Gbps) / Network bandwidth measurement
  • 存储设备营销 (硬盘、U盘) / Storage device marketing
  • 数据传输速度描述 / Data transfer speed description
  • 互联网流量统计 / Internet traffic statistics

传统单位定义 / Conventional Unit Definitions

单位 / Unit 等于 / Equals (Power) 字节 / Bytes
1 kilobyte (KB) 10³ 1,000
2 megabyte (MB) 10⁶ 1,000,000
3 gigabyte (GB) 10⁹ 1,000,000,000
4 terabyte (TB) 10¹² 1,000,000,000,000
5 petabyte (PB) 10¹⁵ 1,000,000,000,000,000

计算示例 / Calculation Examples

  • 1 KB = 1,000 bytes
  • 1 MB = 1,000 KB = 1,000,000 bytes
  • 1 GB = 1,000 MB = 1,000,000,000 bytes

注意事项 / Important Notes

  • 存储容量可能出现偏差 / Storage capacity may show discrepancy
  • 1TB硬盘实际可用空间约 931GB / 1TB HDD has ~931GB usable space
  • 这是因为 1000 vs 1024 的差异 / This is due to 1000 vs 1024 difference
