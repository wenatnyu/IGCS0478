图像的位图表示 / Bitmap Image Representation

基本概念 / Basic Concepts

  • 位图图像由像素组成 / Bitmap images are made up of pixels
  • 在计算机中存储为一系列二进制数字 / Stored as series of binary numbers
  • 每个像素包含颜色和位置信息 / Each pixel contains color and position information
  • 图像质量取决于像素的数量和颜色信息 / Quality depends on number of pixels and color info

存储结构与格式 / Storage & Formats

存储结构 / Storage Structure

  1. 图像头信息: 尺寸、格式等 / Header info: size, format, etc.
  2. 像素数据: 逐个像素的颜色值 / Pixel data: color values for each pixel
  3. 存储方式: 按行或列存储 / Stored row by row or column

常见格式 / Common Formats

  • BMP: 未压缩，大文件 / Uncompressed, large files
  • JPEG: 有损压缩，适合照片 / Lossy compression, good for photos
  • PNG: 无损压缩，支持透明 / Lossless compression, supports transparency

像素概念 / Pixel Concept

  • Pixel = Picture Element (图像元素)
  • 图像中最小的单位 / Smallest unit in image
  • 以网格形式排列 / Arranged in a grid
  • 坐标系统: (x, y)位置 / Coordinate system: (x, y) position

位图特点 / Bitmap Characteristics

  • 固定分辨率，放大会失真 / Fixed resolution, loses quality when enlarged
  • 像素化: 放大后显示马赛克效果 / Pixelation: mosaic effect when enlarged
  • 适合照片和复杂图像 / Suitable for photos and complex images
  • 文件大小与分辨率、色彩成正比 / File size proportional to resolution & colors
