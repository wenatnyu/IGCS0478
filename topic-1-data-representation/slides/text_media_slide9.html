色彩深度和图像分辨率 / Color Depth & Image Resolution

色彩深度 / Color Depth

  • 表示每种颜色位数的称为色彩深度
    Number of bits per color
  • 8位 = 256色 (2⁸=256)
  • 1位 = 2色 (2¹=2)
  • 增加色彩深度会增加文件大小
    More depth = larger file size

常见色彩深度 / Common Depths

  • 1-bit: 2色 (B&W)
  • 8-bit: 256色 (Grayscale/Indexed)
  • 16-bit: 65,536色
  • 24-bit: 16.7M色 (True Color)
  • 32-bit: 24位 + Alpha (Transparency)

图像分辨率 / Image Resolution

指构成图像的像素数量。
The number of pixels in an image.
