HTML颜色定义 / HTML Color Definition

  • 在超文本标记语言(HTML)中定义颜色 / Used to define colors in HTML
  • 使用 #RRGGBB 格式 / Uses #RRGGBB format
  • 每组代表红、绿、蓝三原色的强度 / Each pair represents R, G, B intensity

实际示例 / Examples

  • #FF0000 = 纯红色 / Pure Red
  • #00FF00 = 纯绿色 / Pure Green
  • #0000FF = 纯蓝色 / Pure Blue
  • #FFFFFF = 白色 / White
  • #000000 = 黑色 / Black
