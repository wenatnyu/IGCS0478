为什么使用十六进制 / Why Use Hexadecimal

比二进制更易于编写代码，速度更快，错误更少 Easier to write code than binary, faster, fewer errors
提供更短的二进制表示，更易于人类理解 Provides shorter binary representation, easier for humans to understand
4个二进制位可以用1个十六进制位表示 4 binary bits can be represented by 1 hexadecimal digit
在计算机系统中作为二进制的可读形式 Acts as readable form of binary in computer systems
