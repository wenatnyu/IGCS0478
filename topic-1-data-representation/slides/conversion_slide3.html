二进制转十进制 / Binary to Denary Conversion

方法说明 / Method Description

权重位置法 / Positional Weight Method

  • 使用权重表：128, 64, 32, 16, 8, 4, 2, 1 Use weight table: 128, 64, 32, 16, 8, 4, 2, 1
  • 将2的幂次从右到左排列 Powers of 2 from right to left

步骤说明 / Step-by-Step Process

  1. 将二进制数字放入权重表格中 Place binary digits in positional weight table
  2. 找到所有为 1 的位置 Identify all positions with value 1
  3. 将对应的权重值相加 Add corresponding weight values

详细示例 / Detailed Example

转换 / Convert: 11101110₂ → ?₁₀

位置/Position 7654 3210
权重/Weight 128643216 8421
二进制/Binary 1 1 1 0 1 1 1 0
计算/Calculation 128×164×132×116×0 8×14×12×11×0
结果/Result 12864320 8420

最终计算 / Final Calculation

128 + 64 + 32 + 0 + 8 + 4 + 2 + 0 = 238₁₀

关键要点 / Key Points

  • 只有为1的位才参与计算 Only positions with 1 participate in calculation
  • 最左位是MSB（最高有效位） Leftmost bit is MSB (Most Significant Bit)
  • 权重值为2的幂次 Weight values are powers of 2
