十六进制与十进制互转 / Hexadecimal ↔ Denary Conversion

核心方法 / Core Method

通过二进制作为中介 / Using Binary as Intermediate 所有转换都依赖二进制 / All conversions rely on binary

方向一：十六进制 → 十进制
Direction 1: Hexadecimal → Denary

Hexadecimal → Binary → Denary

示例 / Example: 2F16 → ?10

步骤 1: 2F16 → 二进制 / Binary

2 = 0010, F(15) = 1111
⇒ 2F16 = 001011112

步骤 2: 001011112 → 十进制 / Denary
权重/Weight 32168421
二进制/Binary 101111
计算/Calc 3208421

32 + 8 + 4 + 2 + 1

结果 / Result: 4710

方向二：十进制 → 十六进制
Direction 2: Denary → Hexadecimal

Denary → Binary → Hexadecimal

示例 / Example: 4710 → ?16

步骤 1: 4710 → 二进制 / Binary (除以2法)
除法/Division商/Quotient余数/Remainder
47 ÷ 2231
23 ÷ 2111
11 ÷ 251
5 ÷ 221
2 ÷ 210
1 ÷ 201

从下往上读取 / Read upwards: 4710 = 1011112

步骤 2: 1011112 → 十六进制 / Hex (4位分组) 
分组 / Grouping (from right to left):
101111 → 0010 | 1111 (pad with zeros)
转换 / Convert:
0010 → 2
1111 → 15 → F

结果 / Result: 2F16

关键要点 / Key Points

  • 必须掌握两个基本转换
    Must master: Binary↔Denary & Binary↔Hex
  • 二进制是所有转换的桥梁
    Binary is the bridge for all conversions
  • 熟练记住 A-F 对应 10-15
    Memorize A-F corresponds to 10-15
