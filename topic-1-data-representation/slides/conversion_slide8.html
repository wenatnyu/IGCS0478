转换方法总结 / Conversion Methods Summary

Topic 1.2: Number Conversions - 数字转换掌握

1

二进制 ↔ 十进制

Binary ↔ Denary

二进制→十进制: 权重位置法 (128, 64, 32, 16, 8, 4, 2, 1) Binary→Denary: Positional weight method
十进制→二进制: 除以2取余法，从下往上读取 Denary→Binary: Division by 2, read remainders bottom to top
2

十六进制 ↔ 二进制

Hexadecimal ↔ Binary

十六进制→二进制: 1位十六进制 = 4位二进制 (直接对应) Hex→Binary: 1 hex digit = 4 binary digits (direct mapping)
二进制→十六进制: 4位一组, 从右向左, 不足补0 Binary→Hex: Group by 4 from right to left, pad with zeros
3

十六进制 ↔ 十进制

Hexadecimal ↔ Denary

两步转换法: 通过二进制作为中介 Two-step method: Using binary as intermediate

Hex ➔ Binary ➔ Denary | Denary ➔ Binary ➔ Hex

实用技巧 / Practical Tips

  • 必须记住: A=10, B=11, C=12, D=13, E=14, F=15 Must memorize: A=10, B=11, C=12, D=13, E=14, F=15
  • 常用权重: 1, 2, 4, 8, 16, 32, 64, 128, 256 Common weights: 1, 2, 4, 8, 16, 32, 64, 128, 256
  • 分组技巧: 二进制从右向左数, 4位一组 Grouping tip: Count binary from right to left, 4 bits per group
  • 检查方法: 反向转换验证结果 Checking method: Reverse conversion to verify results
