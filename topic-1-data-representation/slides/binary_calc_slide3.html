二进制加法规则 / Binary Addition Rules

核心原理 / Core Principle

二进制加法与十进制不同，因为二进制中不存在数字2。
Binary addition differs from denary as the digit 2 doesn't exist in binary.

四个基本规则 / Four Basic Rules

规则 1 / Rule 1: 0 + 0 = 0

  • 两个零相加等于零 / Two zeros equal zero
  • 无进位产生 / No carry generated

规则 2 / Rule 2: 1 + 0 = 1 或 0 + 1 = 1

  • 一加零等于一 / One plus zero equals one
  • 无进位产生 / No carry generated

规则 3 / Rule 3: 1 + 1 = 0 (带进位1)

  • 两个一相加等于零，但产生进位 / Two ones equal zero but generate carry
  • 关键：进位传递到下一位 / Key: Carry propagates to next position

规则 4 / Rule 4: 1 + 1 + 1 = 1 (带进位1)

  • 三个一相加(含上一位的进位) / Three ones (including carry from previous)
  • 结果为1，同时产生进位 / Result is 1, also generates carry

重要要点 / Key Points

  • 进位传播是关键 / Carry propagation is crucial
  • 从右向左计算 / Calculate from right to left
  • 每一位都可能产生进位 / Each bit can generate a carry
  • 进位必须传递给下一位 / Carry must be passed to the next position

简单示例 / Simple Example

1 0 1 (进位/Carry) 1 1 0 (A) + 1 0 1 (B) --------- 0 1 1 (结果/Result)
