MAC地址 / MAC Addresses

概念解释 / Concept

  • 媒体访问控制(MAC)地址 / Media Access Control (MAC) address
  • 网络上唯一标识设备的号码 / Number that uniquely identifies a device on a network
  • 通常显示为6组十六进制数字 / Usually displayed as 6 groups of hexadecimal digits

格式示例 / Format Example

  • 标准格式 / Standard: 00:1B:44:11:3A:B7
  • 另一种格式 / Alternative: 00-1B-44-11-3A-B7
  • 每组: 2个十六进制数字 = 8位 / Each group: 2 hex digits = 8 bits

实际应用 / Practical Application

  • 网络设备识别 / Network device identification
  • 以太网帧传输 / Ethernet frame transmission
  • 本地网络通信 / Local network communication
