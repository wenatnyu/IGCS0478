什么是文件类型？ / What are File Types?

文件类型定义了数据的存储和编码方式 / File types define how data is stored and encoded
不同类型适用于不同的应用场景 / Different types suit different application scenarios

MIDI

MIDI

音乐指令格式
Musical Instruction Format

存储音乐指令而非音频数据 Stores musical instructions, not audio data

MP3

MP3

压缩音频格式
Compressed Audio Format

有损压缩，大幅减小文件大小 Lossy compression, significantly reduces file size

MP4

MP4

多媒体容器格式
Multimedia Container Format

可存储音频、视频、图像等多种媒体 Can store audio, video, images and other media
