二进制补码 / Two's Complement

补码概念 / Concept

二进制补码是一种用于在二进制中表示 负值 的方法。
Two's complement is a method used to represent negative values in binary.

关键变化 / Key Changes

在这里，MSB (最高有效位) 的权重从 +128 变为 -128。
Here, the MSB (Most Significant Bit) weight changes from +128 to -128.

注意 / Note: 最高位 (MSB) 是 负权重 (negative weight)。

补码的优势 / Advantages

  • 统一的加法规则 / Unified addition rules
    正数和负数使用相同的加法电路 / Same circuits for positive and negative numbers.
  • 唯一的零表示 / Unique zero representation
    只有一个零 (00000000) / Only one zero representation.
  • 简化的电路设计 / Simplified circuit design
    减少硬件复杂性 / Reduces hardware complexity.

MSB 作为符号位 / MSB as Sign Bit

  • MSB = 0: 表示正数 / Represents a positive number.
  • MSB = 1: 表示负数 / Represents a negative number.

表示示例 / Representation Examples

  • -1: 11111111
    -128 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = -1
  • -70: 10111010
    -128 + 32 + 16 + 8 + 2 = -70
  • -128: 10000000
    -128 + 0 = -128 (最小值/Min)
  • +127: 01111111
    0 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = +127 (最大值/Max)
