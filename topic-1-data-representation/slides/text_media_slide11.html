知识总结 / Summary

1

文本表示 / Text Representation

  • 字符集将字符映射为二进制代码
  • ASCII: 7位, 128字符, 基础英文
  • Unicode: 最多32位, 支持所有语言
2

声音数字化 / Sound Digitization

  • 模拟信号通过采样转为数字信号
  • 采样率: 每秒采样次数 (Hz)
  • 采样分辨率: 每样本位数 (位深)
3

图像表示 / Image Representation

  • 位图图像由像素组成
  • 图像分辨率: 像素数量 (宽×高)
  • 色彩深度: 每像素颜色位数
4

质量与文件大小 / Quality vs File Size

  • 更高质量 = 更大文件
  • 需根据用途平衡选择
  • 压缩技术可减小文件大小

实际应用原则 / Practical Principles

  • 网络传输: 低分辨率，快速加载
  • 存档保存: 高分辨率，无损质量
  • 移动设备: 中等参数，平衡性能

文件大小计算 / File Size Calculation

声音 / Audio:

Size = Rate × Resolution × Time × Channels

图像 / Images:

Size = Width × Height × Color Depth

总结结论 / Final Conclusions

  • 数字化是多媒体处理的基础
  • 参数选择影响质量和效率
  • 理解原理有助于优化应用
