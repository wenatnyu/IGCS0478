文本表示概述

Text Representation Overview

  • ✓

    为了以二进制表示文本, 计算机使用字符集。

    To represent text in binary, computers use character sets.

  • ✓

    字符集是字符及其对应二进制代码的集合。

    Character sets are collections of characters and corresponding binary codes.

  • ✓

    每个字符都有唯一的数值代码。

    Each character has a unique numeric code.

  • ✓

    计算机将这些数值转换为二进制存储。

    Computers convert these numbers to binary for storage.

字符 'A' → ASCII 65 → 二进制 01000001

字符 'a' → ASCII 97 → 二进制 01100001
