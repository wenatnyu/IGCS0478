课程总结 / Course Summary

Topic 1.1: Number Systems - 数字系统基础

知识要点总结 / Key Knowledge Summary

1. 三种数字系统 / Three Number Systems

  • 十进制 (Base 10): 0-9, 日常使用 / Denary (Base 10): 0-9, daily use
  • 二进制 (Base 2): 0,1, 计算机语言 / Binary (Base 2): 0,1, computer language
  • 十六进制 (Base 16): 0-9, A-F, 编程应用 / Hexadecimal (Base 16): 0-9, A-F, programming use

2. 核心概念 / Core Concepts

  • 二进制与电子开关 / Binary and electronic switches (ON/OFF)
  • MSB 最高有效位 / MSB (Most Significant Bit)
  • 4位二进制 = 1位十六进制 / 4 binary bits = 1 hex digit
  • 数字系统之间的转换 / Conversions between number systems

3. 实际应用 / Practical Applications

  • HTML颜色代码 / HTML color codes
  • MAC地址 / MAC addresses
  • 内存地址 / Memory addresses
  • 机器代码 / Machine code
  • 调试工具 / Debugging tools

学习目标清单 / Learning Objectives Checklist

  • ✅理解计算机使用二进制的原因 / Understand why computers use binary
  • ✅掌握三种数字系统的概念 / Master three number systems concepts
  • ✅认识十六进制的优势 / Recognize hexadecimal advantages
