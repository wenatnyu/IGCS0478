基本存储单位 / Basic Storage Units

最小存储单位 / Smallest Storage Units

  • BIT (位)

    • 一个二进制数字 / A binary digit
    • 只能是0或1 / Can only be 0 or 1
    • 计算机存储的最小单位 / Smallest unit of computer storage

  • NIBBLE (半字节)

    • 4个位 / 4 bits (1 Nibble = 4 Bits)
    • 可以表示0-15的数值 / Can represent values 0-15
    • 一个十六进制数字 / One hexadecimal digit

  • BYTE (字节)

    • 8个位 / 8 bits (1 Byte = 8 Bits = 2 Nibbles)
    • 可以表示0-255的数值 / Can represent values 0-255
    • 一个字符的存储大小 / Storage size for one character

重要关系 & 应用 / Key Relationships & Applications

  • 1 Byte = 8 Bits, 1 Nibble = 4 Bits, 1 Byte = 2 Nibbles
  • 字节用于测量内存大小 / Byte is used to measure memory size
  • 一个ASCII字符 = 1 Byte / One ASCII character = 1 Byte
  • 一个中文字符 (Unicode) = 2-4 Bytes / One Chinese character = 2-4 Bytes
