Topic 1.2 Number Conversions - 数字转换

1

掌握二进制与十进制之间的转换

Convert between positive binary and positive denary numbers

  • 二进制 → 十进制 (权重法) / Binary → Denary (positional weights)
  • 十进制 → 二进制 (除以2取余法) / Denary → Binary (division by 2)
2

掌握十六进制与十进制之间的转换

Convert between positive hexadecimal and positive denary numbers

  • 通过二进制作为中介 / Through binary as intermediate step
  • 十六进制 ↔ 二进制 ↔ 十进制 / Hexadecimal ↔ Binary ↔ Denary
3

掌握十六进制与二进制之间的转换

Convert between positive hexadecimal and positive binary numbers

  • 十六进制 → 二进制 (1位十六进制 = 4位二进制) / Hex → Binary (1 hex = 4 binary)
  • 二进制 → 十六进制 (4位一组) / Binary → Hex (group by 4)

转换方法预览 / Conversion Methods Preview

  • 权重表法 / Positional weight method: 128, 64, 32, 16, 8, 4, 2, 1
  • 除以2取余法 / Division by 2 with remainders
  • 4位分组法 / 4-bit grouping method
  • 查表法 / Lookup table method: A=10, B=11..F=15
