IP地址 / IP Addresses

概念介绍 / Concept

  • 互联网协议地址 / Internet Protocol addresses
  • IPv6地址使用十六进制表示 / IPv6 addresses use hexadecimal representation
  • 128位地址, 可提供大量唯一地址 / 128-bit address, provides vast number of unique addresses

IPv6 格式 / Format

  • 标准格式 / Standard:
    2001:0db8:85a3:0000:0000:8a2e:0370:7334
  • 简化格式 / Simplified:
    2001:db8:85a3::8a2e:370:7334
  • 8组, 每组4个十六进制数字 / 8 groups, 4 hex digits each
  • 用冒号分隔 / Separated by colons

实际应用 / Application

  • 现代互联网通信 / Modern internet communication
  • 网络配置和管理 / Network configuration and management
  • 物联网设备寻址 / IoT device addressing

十六进制 → 二进制 / Hex → Binary
