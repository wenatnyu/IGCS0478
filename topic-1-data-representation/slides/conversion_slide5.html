十六进制转二进制 / Hexadecimal to Binary Conversion

方法说明/ Method

  • 直接对应法 / Direct Mapping 1 hex digit = 4 binary digits
  • 使用对应表 / Use Table Replace each hex digit with its 4-bit binary equivalent.

对应表/ Table

01234567
00000001001000110100010101100111
89ABCDEF
10001001101010111100110111101111

详细示例/ Detailed Example

Convert: 21FD16 → ?2

分步转换 / Step-by-Step Conversion:

  • 2→0010 查表得 / Look up: 2 = 00102
  • 1→0001 查表得 / Look up: 1 = 00012
  • F→1111 查表得 / Look up: F = 11112
  • D→1101 查表得 / Look up: D = 11012

组合结果 / Combining Results:

0010 0001 1111 1101

最终结果 / Final Result:

21FD16 = 00100001111111012

步骤说明/ Steps

  1. 1
    分离每个十六进制位Separate each hex digit.
  2. 2
    转换为4位二进制Convert each to 4-bit binary.
  3. 3
    按顺序组合Combine groups in order.

重要提示/ Tips

  • 每个十六进制位必须是4位二进制。Each hex digit MUST be 4 binary digits.
  • 不足4位的在左侧补0 (例如 7 → 0111)。Pad with zeros on the left if needed.

快速查表/ Quick Ref

  • A=10, B=11, C=12, D=13, E=14, F=15
  • 常用: 8=1000, C=1100, F=1111
