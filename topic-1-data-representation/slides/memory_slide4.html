IECB系统 / IECB System (Binary-based)

系统特点 / System Characteristics

  • 也称为二进制系统 / Also called Binary System
  • 使用 2&sup_n; 作为基础 / Uses 2&sup_n; as base
  • 每个单位 = 1024 × 前一个单位 / Each unit = 1024 × previous unit
  • 计算机中最常用的系统 / Most commonly used system in computers

关键特点 / Key Features

  • 使用“i”后缀 (kibi, mebi, gibi) / Uses 'i' suffix (kibi, mebi, gibi)
  • 精确反映计算机的二进制本质 / Accurately reflects computer's binary nature
  • 防止与十进制系统混淆 / Prevents confusion with decimal system
  • 由国际电工委员会(IEC)标准化 / Standardized by IEC

为什么重要 / Why Important

  • 计算机系统内部使用的实际系统 / Actual system used internally by computer systems
  • 内存、处理器缓存等使用此系统 / Memory, processor cache, etc. use this system
  • 精确的技术计算必须 / Required for precise technical calculations

IECB单位定义 / IECB Unit Definitions

  • 1 kibibyte (KiB)= 210= 1,024 bytes
  • 1 mebibyte (MiB)= 220= 1,048,576 bytes
  • 1 gibibyte (GiB)= 230= 1,073,741,824 bytes
  • 1 tebibyte (TiB)= 240= 1,099,511,627,776 bytes
  • 1 pebibyte (PiB)= 250= 1,125,899,906,842,624 bytes

计算示例 / Calculation Examples

  • 1 MiB = 1,024 KiB = 1,048,576 bytes
  • 1 GiB = 1,024 MiB = 1,073,741,824 bytes
