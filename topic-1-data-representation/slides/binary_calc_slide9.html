知识总结 / Knowledge Summary

Topic 1.3: Binary Calculations - 二进制计算精通

四大核心技能 / Four Core Skills

  • 1. 二进制加法 / Binary Addition

    • 基本规则: 0+0=0, 1+0=1, 1+1=0(进位), 1+1+1=1(进位)

    Basic rules: 0+0=0, 1+0=1, 1+1=0(carry), 1+1+1=1(carry)

    • 8位计算: 从右向左，处理进位传播

    8-bit calculation: Right to left, handle carry propagation

  • 2. 溢出检测 / Overflow Detection

    • 溢出条件: 结果超过8位或大于255

    Overflow condition: Result exceeds 8 bits or greater than 255

    • 溢出指示: 第9位（最左位）为溢出位

    Overflow indication: 9th bit (leftmost) is overflow bit

    • 意义: 内存空间不足存储结果

    Meaning: Insufficient memory space to store result

  • 3. 逻辑移位 / Logical Shifts

    • 左移位: 向左移动，右侧填0，相当于×2

    Left shift: Move left, fill 0 on right, equivalent to ×2

    • 右移位: 向右移动，左侧填0，相当于÷2

    Right shift: Move right, fill 0 on left, equivalent to ÷2

  • 4. 二进制补码 / Two's Complement

    • 用途: 表示负数，范围-128到+127

    Purpose: Represent negative numbers, range -128 to +127

    • MSB: 作为符号位，权重为-128

    MSB: Acts as sign bit, weight is -128
