Section 5.3 - Threats & Solutions

5.3.1 Cyber Security Threats / 网络威胁

Keeping data safe is extremely important (personal & commercial data).

Main Threats Covered:

  • Brute force attacks (暴力破解)
  • Data interception (数据拦截)
  • DDoS attacks (分布式拒绝服务攻击)
  • Hacking (黑客攻击)
  • Malware (恶意软件)
  • Phishing & Pharming (网络钓鱼与嫁接)
  • Social Engineering (社交工程)

Brute Force & Interception / 暴力破解与拦截

Brute Force Attacks

Hackers systematically try all combinations of passwords.

Defense: Strong, long passwords & limit login attempts.

Data Interception

Stealing data by tapping into wired or wireless links (Packet Sniffing).

Defense: Encryption (WEP/WPA) & Firewalls.

DDoS Attacks / 分布式拒绝服务攻击

Distributed Denial of Service

Flooding a server with useless traffic to prevent legitimate access.

Mechanism:
- Attacker sends thousands of requests.
- Server becomes overloaded.

Signs: Slow network, inability to access websites.

Malware Types / 恶意软件类型

Malware Definitions / 恶意软件详解

Type Description
Virus Replicates with intention of deleting/corrupting files. Needs active host program.
Worm Standalone; replicates to spread to other computers. Exploits network weaknesses.
Trojan Horse Disguised as legitimate software. Replaces good software to cause harm.
Spyware Monitors user activity (e.g., key logging) and sends data back to hackers.
Ransomware Encrypts user data; demands money (ransom) for decryption key.

Phishing vs Pharming / 钓鱼与嫁接

Phishing (网络钓鱼)

Legitimate-looking emails trick users into clicking links.

Defense: Check URLs, Don't click links.

Pharming (DNS嫁接)

Malicious code redirects browser to a fake website (DNS poisoning).

Defense: Check certificate (Green Padlock).

Social Engineering / 社交工程

Manipulating people into breaking security procedures (Fear, Curiosity, Trust).

1. Identify Victim
2. Target (Email/Call)
3. Execute Attack
4. Remove Traces

Solutions: Biometrics / 生物识别

Access Control

Using unique human characteristics for authentication.

  • Fingerprint (指纹)
  • Retina Scans (视网膜)
  • Face Recognition (人脸)
  • Voice Recognition (声纹)

Firewalls & Proxy / 防火墙与代理

Firewall (防火墙)

Filters traffic between user and internet. Can be software or hardware.

Proxy Server (代理服务器)

Intermediary between user and web server.

  • Hides user IP (Hidden Identity)
  • Filters traffic
  • Caches websites (Speed)

SSL/TLS Encryption / 安全协议

SSL/TLS encrypts data to ensure secure transmission.
