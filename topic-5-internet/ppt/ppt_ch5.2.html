5.2.1 Digital Currency / 数字货币

Definition / 定义

  • Exists purely in digital format. No physical form.
    仅以数字形式存在，无实体。
  • Accepted form of payment for goods/services.
    公认的支付方式。
  • Money exists as data on a computer system.
    金钱即计算机系统中的数据。

Fiat Currency (法币): Conventional currency ($/£/€) backed by governments/banks.
由政府和银行支持的传统货币。

Centralized Banking / 中心化银行系统

Traditional digital currency relies on a central banking system.
传统数字货币依赖中央银行系统。

Problem: Maintaining confidentiality and security. (保密性与安全性问题)

Cryptocurrency & Decentralization / 加密货币

Cryptocurrency overcomes centralization issues by using cryptography and decentralization.
加密货币通过密码学和去中心化解决了中心化问题。

Traditional vs Cryptocurrency / 对比

Traditional Digital Currency (传统) Cryptocurrency (加密货币)
Regulated by Central Banks & Governments.
受央行和政府监管。		 No State Control. Rules set by community.
无国家控制，由社区制定规则。
Transactions are Private to the bank.
交易对银行私有。		 Transactions are Publicly Available (tracked).
交易公开可见（可追踪）。
Centralized Server.
中心化服务器。		 Blockchain Network (Secure).
区块链网络（安全）。

5.2.2 Blockchaining / 区块链技术

Concept (概念)

A decentralized database consisting of interconnected computers not connected to a central server.

All transaction data is stored on all computers in the network.

去中心化数据库。所有交易数据存储在网络中的所有计算机上。

Security (安全)

When a transaction occurs, all computers get a copy.

Cannot be changed without consent of all network members. Prevents hacking.

交易发生时所有节点同步。未经所有成员同意不可篡改。

Uses: Cryptocurrency, Smart Contracts, Research, Politics, Education.

How Blockchain Works: The Block / 区块结构

Whenever a transaction takes place, a new Block is created containing three parts:

  1. Data: Sender, Recipient, Amount.
  2. Hash Value: Unique fingerprint (SHA 256).
  3. Previous Hash: Points back to previous block.

The Blockchain Mechanism / 链式结构

Immutability: If Block 2 is changed, its Hash changes. Block 3's "Previous Hash" link breaks, invalidating the chain.
不可篡改性：如果修改区块2，其哈希值改变。区块3的“前一哈希”链接断裂，导致链条无效。

Why is it Secure? / 安全机制

1. Proof-of-Work (工作量证明)

It takes time (e.g., 10 mins) to calculate the proof for each block.

Prevents hackers from quickly recreating a new chain.

计算区块证明需要时间，防止黑客快速重建新链。

2. Miners (矿工)

Special users who calculate blocks and get commission.

They check blocks for correctness.

负责计算和验证区块的特殊用户，获得佣金。

3. Distributed (分布式)

To hack the system, you must attack every single block on every computer at the same time.

必须同时攻击每台计算机上的每个区块才能破解。

4. Genesis Block (创世区块)

The first block in the chain (Block 1). It has no previous hash (0000).

链中的第一个区块，没有前一个哈希值。
