Concept: Internet vs WWW / 概念辨析

The Internet (Hardware)

"INTERconnected NETwork". A worldwide collection of interconnected networks. It is a concept relying on physical infrastructure (cables, routers, satellites).

互联网是基于物理基础设施（硬件）的全球互联网络。

The WWW (Software)

A massive collection of multimedia web pages accessed via the internet using specific software (browsers).

万维网是通过互联网访问的多媒体网页集合（软件应用）。

Key Differences / 关键区别 (Table 5.1)

Internet (互联网)
Nature: A worldwide collection of interconnected networks and devices (Hardware).
互联网络和设备的集合（硬件）。
Protocols: Uses TCP (Transmission Control Protocol) and IP (Internet Protocol).
使用 TCP 和 IP 协议。
World Wide Web (万维网)
Nature: A collection of multimedia web pages and information on websites (Software).
多媒体网页和信息的集合（软件）。
Protocols: Uses HTTP/HTTPS (Hypertext Transfer Protocol) and URLs.
使用 HTTP 协议和 URL。

Uniform Resource Locators (URLs) / 统一资源定位符

URLs are text addresses used to access websites. A URL structure is defined as:

https:// Protocol
www. Host
hoddereducation Domain Name
.co Type
.uk Country
/ict Path/File

Protocol (协议)

http or https. Rules for transferring files.

Domain Type (域名类型)

.com (commercial), .org (organization), .gov (government). Sometimes includes country code (.uk, .cn).

Protocols & Security / 协议与安全

HTTP

Hypertext Transfer Protocol

The standard set of rules obeyed when transferring files across the internet. Data is sent in plain text.

超文本传输协议：互联网文件传输的标准规则。数据明文传输。

HTTPS (Secure)

Secure Encryption (SSL/TLS)

The 'S' stands for Secure. A padlock icon appears in the browser. Data is encrypted and safe from hackers.

加密传输：'S' 代表安全。浏览器显示锁型图标。数据被加密。

Web Browser Features / 浏览器功能

Browsers interpret (translate) HTML code to display content (text, images, video) to the user.

  • Home Page: The default starting page.
    主页
  • History: Log of visited websites.
    历史记录
  • Bookmarks: Stored favorite URLs.
    书签/收藏夹
  • Cache: Stores recent data for faster loading.
    缓存 (加速加载)
  • Cookies: Stores user preferences.
    Cookie 数据

DNS: Domain Name System / 域名系统

The "Phonebook" of the Internet

Web servers are identified by IP Addresses (e.g., 107.162.140.19).

Humans prefer names (www.google.com). DNS servers contain a database that maps these Domain Names to IP Addresses.

Benefit: Users do not need to memorize complex numeric IP addresses.

优点：用户无需记忆复杂的数字 IP 地址。

How DNS Works (Step-by-Step) / 解析过程

01

Request (请求)

User types URL. Browser asks DNS server for IP.

02

Search (查找)

DNS Server 1 checks database. If not found, asks DNS Server 2.

03

Return & Connect (返回与连接)

IP (107.162.140.19) is sent back to user. Browser connects to server.

Cookies: How they work / Cookie 机制

Definition / 定义

Small text files stored on a user's computer. Each cookie is a "Look-up table" of key-value pairs.

{ surname: "Jones" }
{ currency: "USD" }
{ language: "EN" }

Purpose / 用途

  • Maintain user preferences. (用户偏好)
  • Track browsing activity. (追踪浏览行为)
  • "Customers who bought X also bought Y".

Cookies are generated by the Web Server but stored on the User's Browser.

Types of Cookies / Cookie 类型

Session Cookies (会话)

Storage: Temporary Memory (RAM).

Lifespan: Deleted when browser closes.

Use Case: Virtual Shopping Basket.

不收集个人信息，仅用于维持当前会话（如购物车）。

Persistent Cookies (持久)

Storage: Hard Drive.

Lifespan: Until expiry date or deleted.

Use Case: Auto-Login (Remember Me).

记住登录状态和偏好设置，即使关闭浏览器也会保留。
