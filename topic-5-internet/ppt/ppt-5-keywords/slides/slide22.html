安全防护措施

Security Measures
Two-Step Verification
两步验证

Authentication requiring two methods of verification to prove user identity.

需要两种验证方法来证明用户身份的身份验证类型。

Patch
补丁

An update for software to improve it and/or remove bugs.

为改进软件和/或删除任何错误而开发的软件更新。

Typo Squatting
错字抢注

Using subtle spelling errors in website addresses to trick users into visiting fake sites.

使用网站地址中的细微拼写错误来欺骗用户访问虚假网站。

Firewall
防火墙

Software/hardware between computer and network that monitors and filters traffic.

位于计算机和外部网络之间的软件或硬件，监控和过滤所有流量。
