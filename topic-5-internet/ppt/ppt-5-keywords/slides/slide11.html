网络安全威胁

Cyber Security Threats
Brute Force Attack
暴力破解攻击

A trial-and-error method to crack passwords by testing all possible combinations.

网络犯罪分子使用的"试错"方法，通过尝试所有可能的组合来破解密码。

Word List
词表

A text file containing words used in brute force attacks.

包含在暴力破解攻击中使用的词汇集合的文本文件。

Data Interception
数据拦截

An attempt to eavesdrop on network transmission using packet sniffing or wardriving.

试图窃听网络传输；通常使用数据包嗅探或接入点映射来拦截数据。

Packet Sniffing
数据包嗅探

A method to examine data packets sent over a network to find their contents.

检查通过网络发送的数据包并查找数据包内容的方法。
