安全防护措施

Security Measures
Access Levels
访问级别

Different levels of access in a system based on user's security level.

计算机系统中根据用户安全级别允许的不同访问级别层次结构。

Anti-Spyware
反间谍软件

Software that detects and removes spyware programs from a system.

检测并删除系统上安装的间谍软件程序的软件。

Authentication
身份验证

Proving user's identity using something they know, have, or are unique to them.

通过用户知道的、拥有的或对其独特的东西来证明用户身份的过程。

Biometrics
生物识别

Authentication using unique human characteristics like fingerprints or voice.

使用独特人类特征（如指纹、声音或视网膜血管模式）的身份验证类型。
