互联网与网页基础

Internet & Web Basics
URL
统一资源定位符

A text-based address for a web page.

网页的文本地址，用于在互联网上唯一标识和定位资源。

HTTPS
安全超文本传输协议

HTTP with extra security (such as SSL) applied.

应用了额外安全措施（如SSL）的HTTP协议，确保数据传输加密。

Hyperlink
超链接

Highlighted text or image that links to further content when clicked.

通过点击激活的高亮文本或图像，链接到其他文本、图像、网页或网站。

DNS
域名服务器

A server that looks up domain names to find IP addresses for web servers.

查找网站域名以找到计算机定位网络服务器所需IP地址的服务器。
