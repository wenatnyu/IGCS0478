恶意软件类型

Types of Malware
Virus
病毒

A program that replicates itself to delete or corrupt files or cause system malfunction.

自我复制的程序，旨在删除或破坏文件或导致计算机系统故障。

Active Host
活动主机

Functioning software that a virus can affect by attaching to or altering its code.

病毒可以通过附加到代码或更改代码来影响的功能软件。

Worm
蠕虫

A stand-alone malware that can self-replicate without an active host or user action.

可以自我复制的独立恶意软件，不需要活动主机或用户操作。

Trojan Horse
特洛伊木马

Malware disguised as legitimate software but containing malicious code.

设计看起来像合法软件但包含可能损害系统的恶意代码的恶意软件。
