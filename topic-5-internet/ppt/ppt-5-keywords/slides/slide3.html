互联网与网页基础

Internet & Web Basics
Internet
互联网

The world-wide interconnection of networks using TCP and IP protocols.

全球网络互联系统；使用TCP和IP协议连接世界各地的计算机网络。

World Wide Web (WWW)
万维网

A massive collection of web pages based on HTTP and HTTPS protocols.

基于超文本传输协议（http和https）的海量网页集合。

(Web) Browser
网页浏览器

Software that connects to DNS to locate IP addresses and interprets HTML web pages.

连接域名服务器定位IP地址的软件；解释HTML网页使用户能阅读文档和观看多媒体。

HTML
超文本标记语言

The language used to design, display and format web pages.

用于设计、显示和格式化网页以及编写http(s)协议的语言。
