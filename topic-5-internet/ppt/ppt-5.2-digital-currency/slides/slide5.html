什么是区块链？
What is Blockchain?

去中心化的分布式数据库，存储网络成员的所有交易记录

A decentralized database storing all transactions across network members

  • 多台计算机连接，无中央服务器 Multiple computers connected, no central server
  • 所有交易数据分布存储 All transaction data is distributed across nodes
  • 新交易同步到所有计算机 New transactions synchronized to all computers
  • 数据透明且不可篡改 Data is transparent and immutable
