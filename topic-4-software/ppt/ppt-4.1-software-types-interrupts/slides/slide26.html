中断服务流程 Interrupt Service Routine (ISR)
1. 接收中断信号
Receive interrupt signal
2. 保存当前任务状态（程序计数器和寄存器）
Save current task status (PC and registers)
3. 加载ISR起始地址到程序计数器
Load ISR start address into Program Counter
4. 执行中断服务程序
Execute interrupt service routine
5. 恢复中断任务状态，继续执行
Restore interrupted task status, continue execution
