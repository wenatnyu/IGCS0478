语言类型定义 Language Type Definition

高级语言
High-level Language
接近人类自然语言和数学表达式的编程语言
Programming language close to natural human language and mathematical expressions
例如 Examples: Python, Java, C++, JavaScript
低级语言
Low-level Language
接近机器语言，直接操作硬件的编程语言
Programming language close to machine code, directly manipulating hardware
例如 Examples: Assembly Language, Machine Code
