汇编语言与汇编器 Assembly Language

汇编语言 Assembly Language
使用助记符（mnemonics）表示机器指令的低级语言
Low-level language using mnemonics to represent machine instructions
汇编器 Assembler
将汇编语言翻译成机器代码的程序
Program that translates assembly language into machine code
助记符示例 Mnemonic Examples
MOV AX, 5 ; Move 5 to AX
ADD AX, 3 ; Add 3 to AX
SUB BX, 2 ; Subtract 2 from BX
JMP LOOP ; Jump to LOOP
CMP AX, BX ; Compare AX and BX
PUSH AX ; Push AX to stack
