Extension - Computer Ports / 计算机端口

USB (Universal Serial Bus)

Structure / 结构

4-wired shielded cable / 4线屏蔽电缆

2 wires: Power and earth / 电源和接地

2 wires: Data transmission / 数据传输

USB Benefits / USB优点

  • ✓

    Auto-detection of devices / 自动检测设备

  • ✓

    Automatic driver loading / 自动加载驱动程序

  • ✓

    Industry standard / 行业标准

  • ✓

    Multiple transmission rates / 多种传输速率

  • ✓

    Backward compatible / 向后兼容

USB Drawbacks / USB缺点

  • ✗

    Limited to <500 Mbps / 限制<500 Mbps

  • ✗

    Max cable length ~5 meters / 最大电缆长度约5米

  • ✗

    Older USB 1.1 may not be supported / 旧版USB 1.1可能不受支持

HDMI (High-Definition Multimedia Interface)

Digital replacement for VGA analog system

VGA模拟系统的数字替代品

HDMI Features / HDMI特性

  • ✓

    Supports audio and video output / 支持音频和视频输出

  • ✓

    Widescreen format (16:9) / 宽屏格式（16:9）

  • ✓

    High resolution (1920×1080 pixels) / 高分辨率（1920×1080像素）

  • ✓

    Fast refresh rate (120 Hz) / 快速刷新率（120 Hz）

  • ✓

    Bandwidth: 10 Gbps / 带宽：10 Gbps
