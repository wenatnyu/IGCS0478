Internet Protocol (IP) Address / IP地址

Private IP / 私有IP: Router assigns private IP to devices on local network / 路由器为本地网络上的设备分配私有IP

Public IP / 公有IP: ISP assigns unique public IP to router for internet access / ISP为路由器分配唯一的公有IP以访问互联网

IPv4 (32 bits / 32位)

Format / 格式: Four groups of 8 bits in decimal / 四组8位十进制

Example / 示例: 254.25.28.77

Problem / 问题: Limited number of addresses / 地址数量有限

IPv6 (128 bits / 128位)

Format / 格式: Eight groups of hexadecimal digits / 八组十六进制数字

Example / 示例: A8FB:7A88:FFF0:0FFF:3D21:2085:66FB:F0FA

Advantages / 优势:

  • ✓
    Removes risk of IP collisions / 消除IP冲突风险
  • ✓
    Built-in authentication checks / 内置身份验证
  • ✓
    More efficient packet routes / 更高效的数据包路由
