3.4.4 Routers / 路由器

Function / 功能

✓

Enable data packets to be routed between different networks (e.g., LAN to WAN)

使数据包能够在不同网络之间路由（例如，LAN到WAN）

✓

Convert data between different protocols

在不同协议之间转换数据

How Routers Work / 路由器工作原理

1

Inspect data packets / 检查数据包

Router receives packets from any device on connected networks

路由器从连接网络上的任何设备接收数据包

2

Use IP addressing / 使用IP地址

Devices on same network share part of IP address

同一网络上的设备共享部分IP地址

Router sends packet to appropriate switch using IP

路由器使用IP将数据包发送到相应的交换机

3

Use MAC addressing / 使用MAC地址

Switch delivers data to correct device using MAC address

交换机使用MAC地址将数据传送到正确的设备

Network Setup / 网络设置

✓

Broadband routers sit behind firewall for protection

宽带路由器位于防火墙后面以提供保护

✓

Can be wired or wireless

可以是有线或无线
