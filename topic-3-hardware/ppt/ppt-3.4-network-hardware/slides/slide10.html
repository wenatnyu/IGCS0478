Extension - Wireless Networks: Wi-Fi vs Bluetooth / 无线网络：Wi-Fi与蓝牙

Bluetooth

Frequency / 频率:

79 channels centered on 2.45 GHz / 79个频道，中心频率2.45 GHz

Technology / 技术:

Spread-spectrum frequency hopping / 扩频跳频

• English: Randomly picks channel to avoid interference

• 中文: 随机选择频道以避免干扰

Range / 范围:

<30 meters / <30米

Security / 安全:

WPAN with key encryption / 带密钥加密的WPAN

Best for / 最适合:

Low bandwidth apps (music to headset) / 低带宽应用（音乐到耳机）

Wi-Fi

Range / 范围:

Up to 100 meters / 最多100米

Speed / 速度:

Faster data transfer than Bluetooth / 比蓝牙更快的数据传输

Security / 安全:

Better security / 更好的安全性

Best for / 最适合:

Full-scale networks / 全规模网络

Access / 访问:

Wireless Access Point (WAP) or 'hot spot' / 无线接入点或“热点”
