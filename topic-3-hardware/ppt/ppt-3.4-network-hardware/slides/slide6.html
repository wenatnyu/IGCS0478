MAC Addresses vs IP Addresses / MAC地址与IP地址对比

Feature / 特性 MAC Address / MAC地址 IP Address / IP地址
Purpose / 用途 Identifies physical address of device on network / 标识网络上设备的物理地址 Identifies global address on internet / 标识互联网上的全局地址
Uniqueness / 唯一性 Unique for device on network / 网络上设备唯一 May not necessarily be unique / 不一定唯一
Assignment / 分配方式 Assigned by manufacturer (part of NIC) / 由制造商分配（NIC的一部分） Dynamic: assigned by ISP using DHCP / 动态：由ISP使用DHCP分配
Changeability / 可变性 Can be UAA or LAA / 可以是UAA或LAA Dynamic IP changes; Static IP doesn't / 动态IP会变；静态IP不变
Usage / 使用场景 Identifies sender's and recipient's devices / 标识发送方和接收方设备 Used in routing operations / 用于路由操作
Size / 大小 48 bits / 48位 32 bits (IPv4) or 128 bits (IPv6) / 32位(IPv4)或128位(IPv6)
Type / 类型 UAA or LAA / UAA或LAA Static or Dynamic / 静态或动态
