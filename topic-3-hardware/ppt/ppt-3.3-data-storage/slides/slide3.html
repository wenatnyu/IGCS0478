Primary Memory vs Secondary Storage / 主存储器与辅助存储

Primary Memory / 主存储器

  • Directly addressable by CPU / CPU可直接访问
  • Volatile (RAM) / 易失性 (RAM)
  • Very fast access / 访问速度极快
  • Limited capacity / 容量有限
  • Examples: RAM, ROM, Cache / 示例: RAM, ROM, 缓存
VS

Secondary Storage / 辅助存储

  • Not directly addressable / CPU不能直接访问
  • Non-volatile / 非易失性
  • Slower access / 访问速度较慢
  • Large capacity / 容量大
  • Examples: HDD, SSD, DVD / 示例: HDD, SSD, DVD
