冯·诺依曼模型中的关键专用寄存器

Key Special-Purpose Registers in Von Neumann Model

寄存器 / Register 缩写 / Abbr. 功能/用途 / Function/Purpose
程序计数器
Program Counter		 PC 存储下一条待取指令的内存地址
Stores address of next instruction to be fetched
内存地址寄存器
Memory Address Register		 MAR 保存当前正在读取或写入的内存位置地址
Holds address of memory location being accessed
内存数据寄存器
Memory Data Register		 MDR 临时存储刚从内存读取或即将写入内存的数据
Temporarily stores data read from or to be written to memory
当前指令寄存器
Current Instruction Register		 CIR 存储当前正在译码和执行的指令
Stores instruction currently being decoded and executed
累加器
Accumulator		 ACC 通用寄存器，临时存储ALU计算结果
General-purpose register for ALU calculation results
