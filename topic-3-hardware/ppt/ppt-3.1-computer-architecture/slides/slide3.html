中央处理器 (CPU) 简介

Introduction to the Central Processing Unit (CPU)

  • CPU是现代计算机系统的基础组件

    The CPU is a foundational component of any modern computer system

  • 物理上是复杂的集成电路

    Physically, a CPU is a complex integrated circuit

  • 主要职责：执行和处理指令和数据

    Primary responsibility: execution and processing of instructions and data

  • 内部组成：控制单元(CU)、算术逻辑单元(ALU)、寄存器

    Internal components: Control Unit (CU), Arithmetic & Logic Unit (ALU), registers
