主存类型与内存特性 | Memory Types & Characteristics

static RAM (SRAM)

静态RAM

Type of RAM chip that uses flip flops and does not need to be constantly refreshed.

使用触发器且不需要不断刷新的RAM芯片类型。

volatile

易失性

Describes memory that loses its contents when the power is turned off.

描述断电时会丢失内容的内存。

refresh

刷新

To recharge every few seconds in order to maintain a charge; devices such as capacitors, for example.

为了保持电荷，每隔几秒钟重新充电；例如电容器等设备。
