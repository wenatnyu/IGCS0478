寄存器 | Registers

CIR (current instruction register)

当前指令寄存器

A register that stores the current instruction being decoded and executed.

存储正在被译码和执行的当前指令的寄存器。

MDR (memory data register)

内存数据寄存器

A register that stores data that has just been read from memory or data that is about to be written to memory.

存储刚从内存读取的数据或即将写入内存的数据的寄存器。

PC (program counter)

程序计数器

A register that stores the address where the next instruction to be read is located.

存储下一条要读取的指令所在地址的寄存器。
