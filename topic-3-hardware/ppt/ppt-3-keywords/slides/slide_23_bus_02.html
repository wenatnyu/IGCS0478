总线类型与数据传输方向 | Bus Types & Data Transfer Direction

控制总线

control bus

从控制单元向所有其他计算机组件传输信号的系统总线。

A system bus that carries signals from the control unit to all other computer components.

单向

unidirectional

只能在一个方向上传输；用于描述数据。

Can only be carried in one direction; used to describe data.

双向

bidirectional

可以在两个方向上传输；用于描述数据。

Can be carried in both directions; used to describe data.
