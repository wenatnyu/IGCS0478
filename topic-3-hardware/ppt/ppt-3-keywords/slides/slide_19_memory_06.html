存储管理 | Storage Management

backing store

后备存储

Secondary storage device (such as HDD or SSD) used to store data permanently when the computer is switched off.

辅助存储设备（如HDD或SSD），用于在计算机关闭时永久存储数据。

address

地址

A label for a memory location which the CPU uses to keep track of data.

CPU用来跟踪数据的内存位置标签。

memory location

内存位置

A numbered location in the memory where values can be stored.

内存中可以存储值的编号位置。
