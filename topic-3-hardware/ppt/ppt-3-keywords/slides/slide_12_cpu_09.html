Basic Input/Output System (BIOS)

基本输入输出系统

A set of programs on firmware used to carry out the initialisation of a computer system during the boot-up process.

固件上的一套程序，用于在启动过程中执行计算机系统的初始化。

embedded system

嵌入式系统

A combination of hardware and software designed to carry out a specific set of functions.

硬件和软件的组合，旨在执行一组特定的功能。
