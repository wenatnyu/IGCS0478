网络地址 | Network Addressing

static IP address

静态IP地址

An IP address that doesn't change.

不会更改的IP地址。

dynamic IP address

动态IP地址

A temporary IP address given to a device every time it logs on to a network.

每次登录网络时分配给设备的临时IP地址。

MAC address

MAC地址

A unique identifier that acts as a network address for a device; takes the form NN-NN-NN-DD-DD-DD where NN is the manufacturer code and DD is the device code.

作为设备网络地址的唯一标识符；采用NN-NN-NN-DD-DD-DD的形式，其中NN是制造商代码，DD是设备代码。
