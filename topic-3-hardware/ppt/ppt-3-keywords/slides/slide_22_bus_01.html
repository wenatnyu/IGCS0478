总线类型 | Bus Types

system buses

系统总线

Connections between the major components in a computer that can carry data, addresses or control signals

计算机中主要组件之间的连接，可以传输数据、地址或控制信号

address bus

地址总线

A system bus that carries addresses around the entire computer system

在整个计算机系统中传输地址的系统总线

data bus

数据总线

A system bus that allows data to be carried from the CPU to memory (or vice versa) or to and from input/output devices

允许数据从 CPU传输到内存（反之亦然）或传输到输入/输出设备的系统总线
