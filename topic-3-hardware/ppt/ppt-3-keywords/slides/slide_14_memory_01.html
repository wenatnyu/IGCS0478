主存类型 | Primary Memory Types

memory

内存

A device in the computer that can be directly accessed by the CPU; there are two types of memory - RAM and ROM; memory is different from storage devices such as hard disk drives

计算机中可由CPU直接访问的设备；有两种类型的内存 - RAM和ROM；内存不同于硬盘驱动器等存储设备

immediate access store (IAS)

立即访问存储器

Memory that holds all the data and programs needed to be accessed by the control unit

保存控制单元需要访问的所有数据和程序的内存
