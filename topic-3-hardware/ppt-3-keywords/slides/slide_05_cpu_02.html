核心组件 | Core Components

Arithmetic & Logic Unit (ALU)

算术逻辑单元

The component in the CPU where all arithmetic and logic operations are carried out.

CPU中执行所有算术和逻辑运算的组件。

Control Unit

控制单元

The CPU component that ensures the synchronised flow of data and programs throughout the computer by sending control signals on the control bus.

CPU组件，通过在控制总线上发送控制信号来确保整个计算机的数据流和程序同步。

Integrated Circuit

集成电路

A chip, usually made from a semi-conductor material, that can carry out the same task as a large circuit made up of individual components.

通常是由半导体材料制成的芯片，可执行与由单个元件组成的大型电路相同的任务。
