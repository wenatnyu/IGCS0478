条形码与二维码 | Barcodes & QR Codes

Barcode
条形码

A series of dark and light lines of different thickness which represents data; the code must be scanned using a laser or LED light source.

一系列不同厚度的深色和浅色线条，用于表示数据；必须使用激光或LED光源扫描代码。

Quick Response (QR) Code
快速响应 (QR) 码

A matrix of dark and light squares which represents data; the pattern can be read and interpreted using a smartphone camera and a QR app.

代表数据的深色和浅色方块矩阵；可以使用智能手机相机和QR应用程序读取和解释图案。
