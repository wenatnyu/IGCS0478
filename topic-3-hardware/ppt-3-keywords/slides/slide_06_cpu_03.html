寄存器 | Registers

register

寄存器

A temporary component in the CPU that can be either general or specific purpose; holds data or an instruction as part of the Fetch-Decode-Execute cycle.

CPU中的临时组件，可用于通用或特定用途；作为取指-译码-执行循环的一部分保存数据或指令。

accumulator (ACC)

累加器

A temporary general-purpose register that stores values in any part of a given operation.

临时通用寄存器，在给定操作的任何部分存储数值。

memory address register (MAR)

内存地址寄存器

A register that stores the address of the memory location currently being read from or written to.

存储当前正在读取或写入的内存位置地址的寄存器。
