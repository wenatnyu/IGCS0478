网络硬件 | Network Hardware

网络接口卡 (NIC)

network interface card (NIC)

A hardware component (a circuit board or chip) needed to allow a device to connect to a network (such as the internet).

允许设备连接到网络（如互联网）所需的硬件组件（电路板或芯片）。

路由器

router

A device that enables data packets to be moved between different networks, for example, connecting a LAN to a WAN.

使数据包能够在不同网络之间移动的设备，例如，将LAN连接到WAN。
