Dynamic vs Static IP Addressing / 动态与静态IP地址

Dynamic IP (DHCP Process / DHCP流程)

  • 1Device requests IP from DHCP server / 设备从DHCP服务器请求IP
  • 2DHCP assigns temporary IP address / DHCP分配临时IP地址
  • 3IP changes each time device connects / 每次连接时IP都会改变
  • 4DNS translates URL to IP address / DNS将URL转换为IP地址

Static IP

  • ✓Permanently assigned by ISP / 由ISP永久分配
  • ✓Consistent location / 位置一致
  • ✓Less bandwidth used / 使用更少的带宽
  • ✓Faster access to resources / 更快访问资源
