Extension - Wired vs Wireless Networks / 有线与无线网络对比

Twisted Pair Cable / 双绞线电缆

Conductor + Insulator + Cable jacket / 导体+绝缘体+电缆护套

Multiple pairs twisted together / 多对绞合在一起

Coaxial Cable / 同轴电缆

Copper wire + Copper mesh + Insulation / 铜线+铜网+绝缘层

Fibre Optic Cable / 光纤电缆

Uses light signals / 使用光信号

Highest speed and security / 最高速度和安全性

Factor / 因素
Wireless / 无线
Wired / 有线
Mobility / 移动性
Easy to expand, devices can move within range / 易于扩展，设备可在范围内移动
Must stay near cables / 必须靠近电缆
Reliability / 可靠性
Subject to interference / 受干扰影响
More reliable and stable / 更可靠和稳定
Speed / 速度
Slower (but improving) / 较慢（但正在改进）
Faster data transfer, no dead spots / 更快的数据传输，无死角
Security / 安全性
Less secure, needs encryption (WEP, WPA2) / 安全性较低，需要加密
More secure, harder to intercept / 更安全，更难拦截
Cost / 成本
Higher setup cost / 更高的设置成本
Cheaper overall / 总体更便宜
Hazards / 危险
No cable hazards / 无电缆危险
Tripping, overheating, disconnection risks / 绊倒、过热、断开风险
Installation / 安装
No cables needed / 不需要电缆
Cables must be installed / 必须安装电缆
