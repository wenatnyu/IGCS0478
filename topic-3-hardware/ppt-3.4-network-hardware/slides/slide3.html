Media Access Control (MAC) Address / MAC地址

Format / 格式: NN-NN-NN-DD-DD-DD

48 bits / 48位: Six groups of hexadecimal digits / 六组十六进制数字

NN-NN-NN
First 6 hex digits / 前6位十六进制
Manufacturer's code / 制造商代码
DD-DD-DD
Last 6 hex digits / 后6位十六进制
Device serial number / 设备序列号

Example / 示例

00-1C-B34F-25-FF

00-1C-B3 = Apple (manufacturer) / 苹果（制造商）

4F-25-FF = Device serial number / 设备序列号

⚠

If NIC card is replaced, MAC address will also change / 如果更换网卡，MAC地址也会改变
