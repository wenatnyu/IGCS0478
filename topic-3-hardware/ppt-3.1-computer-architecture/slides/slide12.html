向内存写入数据的步骤

Steps to Write to Memory

  1. CPU将要写入的数据放入MDR

    CPU places data to be written into MDR

  2. CPU将目标地址放入MAR

    CPU places target address into MAR

  3. 地址和数据分别通过对应总线发送

    Address and data sent via respective buses

  4. CU发送“写入”信号

    CU sends 'write' signal via control bus

  5. 内存控制器将数据存储到指定位置

    Memory controller stores data at specified location

关键区别：与读取操作（先寻址后取数）相反，写入操作是 先准备数据 (MDR)，再 指定地址 (MAR)。

Key Difference: Unlike a read operation (address first, then fetch), a write operation prepares the data first (in MDR) before specifying the address (in MAR).
