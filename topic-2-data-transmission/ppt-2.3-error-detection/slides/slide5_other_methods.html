其他错误检测方法

Other Error Detection Methods

01

校验和 (Checksum)

工作原理 (Working Principle)
发送方计算所有数据块的校验和，与数据一起发送 Sender calculates checksum of all data blocks and sends it with data
接收方重新计算校验和并与接收的校验和对比 Receiver recalculates checksum and compares with received checksum
  • 可检测多位错误 / Can detect multiple bit errors
  • 计算开销较大 / Higher computational overhead
02

回声检查 (Echo Check)

工作原理 (Working Principle)
接收方将接收到的数据发送回发送方 Receiver sends the received data back to the sender
发送方对比原始数据和回声数据进行验证 Sender compares original data with echoed data for verification
  • 简单可靠的验证方法 / Simple and reliable verification method
  • 双倍网络带宽消耗 / Double network bandwidth consumption
03

校验位 (Check Digits)

工作原理 (Working Principle)
从其他所有数字计算得出的额外数字，用于识别数据输入错误，广泛用于标识码系统。 Additional digits calculated from all other digits, used to identify data input errors, widely used in identification systems.
应用实例 (Application Examples)
  • ISBN书号: 用于验证图书编号的正确性 Used to verify book number accuracy
  • 条形码: 用于商品条形码的数据校验 Used for product barcode data validation
  • 身份证号: 用于身份证号码的校验 Used for ID number validation
  • 适合人工输入验证 / Suitable for manual input verification
  • 检测能力有限 / Limited detection capability
