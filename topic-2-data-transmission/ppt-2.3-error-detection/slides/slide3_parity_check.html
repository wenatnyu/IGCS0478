奇偶校验

Parity Checks

1. 基本概念 (Basic Concept)

使用字节中1位的数量进行校验的方法。
A method using the count of '1' bits in a byte for verification.

2. 偶校验 (Even Parity)

整个数据块 (包括校验位) 中1的个数为偶数。
The total number of '1's in the data block (including parity bit) is even.

3. 奇校验 (Odd Parity)

整个数据块 (包括校验位) 中1的个数为奇数。
The total number of '1's in the data block (including parity bit) is odd.

4. 实例演示 (Example)

原始数据: 11010101

偶校验计算: 原始数据中有5个1(奇数), 需添加1作为校验位。
Even Parity: Original data has 5 ones (odd), need to add 1 as parity bit.

结果: 11010101 + 1 = 110101011

奇校验计算: 原始数据中有5个1(已是奇数), 添加0作为校验位。
Odd Parity: Original data has 5 ones (already odd), add 0 as parity bit.

结果: 11010101 + 0 = 110101010

5. 限制性 (Limitation)

两个位同时改变时检测不到错误。
Cannot detect errors when two bits change simultaneously.

例子: 原数据 1100 变成 0011，校验位不变。
