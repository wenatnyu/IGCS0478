USB基本概念和特性

USB Basic Concepts and Characteristics

1. USB定义 / Definition

通用串行总线（USB）是一种异步串行数据传输方法。

Universal Serial Bus (USB) is an asynchronous serial data transmission method.

2. 异步传输 / Asynchronous

数据传输无需时钟同步信号，发送和接收端各自维持时钟频率。

Data transmission requires no clock synchronization signal; devices maintain their own clock frequencies.

3. 串行传输 / Serial

数据按位依次传输，一次只传输一个比特。

Data is transmitted bit by bit sequentially, only one bit at a time.

4. 通用性 / Universality

标准化接口，支持多种设备类型连接。

Standardized interface supporting various device types and connections.
