USB物理结构

USB Physical Structure

1. 四线屏蔽电缆结构 USB uses a four-wire shielded cable design to ensure signal integrity and anti-interference capability. USB采用四线屏蔽电缆设计，确保信号完整性和抗干扰能力。

电源线路 (Power Lines)

红线 (VCC): 提供+5V直流电源 Red wire (VCC): Provides +5V DC power supply

黑线 (GND): 接地线，电路回路 Black wire (GND): Ground wire, circuit return path

数据传输线路 (Data Lines)

白线 (Data-): 负数据信号线 White wire (Data-): Negative data signal line

绿线 (Data+): 正数据信号线 Green wire (Data+): Positive data signal line

4. 屏蔽保护 Outer shielding layer protects internal wires from electromagnetic interference. 外层屏蔽层保护内部线路免受电磁干扰。
