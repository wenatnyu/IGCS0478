数据传输方向类型

Types of Data Transmission Directions

Simplex (单工)

单向传输，数据只能在一个方向传输。

实例：计算机到打印机、电视信号传输

One-way transmission, data flows in only one direction.

Example: Computer to printer, TV signal transmission

Half-duplex (半双工)

双向传输但不同时，同一时间只能一个方向传输。

实例：对讲机、早期以太网

Two-way transmission but not simultaneous, only one direction at a time.

Example: Walkie-talkie, early Ethernet

Full-duplex (全双工)

同时双向传输，数据可以同时在两个方向传输。

实例：宽带电话线、现代以太网

Simultaneous two-way transmission, data flows in both directions at the same time.

Example: Broadband telephone lines, modern Ethernet
