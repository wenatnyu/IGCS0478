数据包结构

Packet Structure

Header（包头）

包含发送方和接收方IP地址、数据包序列号、数据包大小

Contains sender and receiver IP addresses, packet sequence number, packet size

Payload（有效载荷）

包含实际需要传输的数据内容

Contains the actual data to be transmitted

Trailer（尾部）

包含数据包结束标识和错误检查方法

Contains packet end identifier and error checking methods
