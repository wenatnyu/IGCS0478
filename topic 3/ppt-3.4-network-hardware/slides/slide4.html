Types of MAC Addresses

MAC地址类型

UAA (Universally Administered)

通用管理MAC地址

Set by manufacturer at factory (most common type)

由制造商在工厂设置（最常见类型）

Rarely changed by users

用户很少更改

LAA (Locally Administered)

本地管理MAC地址

Can be changed by user or organization

可由用户或组织更改
