多核处理 | Multi-core Processing

core | 核心

A unit on a CPU that consists of the ALU, control unit and registers; there may be more than one core on a CPU.

CPU上由ALU、控制单元和寄存器组成的单元；CPU可能包含多个核心。

dual core | 双核

A CPU that contains two cores.

包含两个核心的CPU。

quad core | 四核

A CPU that contains four cores.

包含四个核心的CPU。
