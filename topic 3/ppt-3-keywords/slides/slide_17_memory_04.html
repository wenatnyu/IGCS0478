内存特性 | Memory Characteristics

FLIP FLOP | 触发器

EN: An electronic circuit that has only two stable states.

中文: 只有两个稳定状态的电子电路。

LATENCY | 延迟

EN: A lag in a system; for example, the time taken to locate a track on a hard disk which depends on the time needed for the disk to rotate to its read-write heads.

中文: 系统中的滞后; 例如, 在硬盘上找到磁道所需的时间, 这取决于磁盘旋转到其读写头所需的时间。

CACHE | 缓存

EN: Temporary memory that uses static RAM to hold frequently used data/instructions by the CPU which improves CPU performance. More generally, cache refers to any storage area used for quick access to frequently used data - other examples include web cache, database cache, DNS cache.

中文: 使用静态RAM的临时内存, 用于保存CPU经常使用的数据/指令, 从而提高CPU性能。更广义地说, 缓存是指用于快速访问常用数据的任何存储区域 - 其他示例包括Web缓存、数据库缓存、DNS缓存。
