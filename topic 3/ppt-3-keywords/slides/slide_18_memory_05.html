虚拟内存 | Virtual Memory

虚拟内存

virtual memory

A memory management system that makes use of secondary storage and software to allow a computer to compensate for shortages in actual physical RAM memory.

利用辅助存储和软件使计算机能够弥补实际物理RAM内存不足的内存管理系统。

磁盘抖动 (HDD)

disk thrashing (HDD)

A problem in a hard disk drive (HDD) caused by excessive movement of the heads due to excessive swapping of data during virtual memory operations.

硬盘驱动器（HDD）中由于虚拟内存操作期间过度交换数据导致磁头移动频率过高而引起的问题。

抖动点

thrash point

The point at which program execution stops because the system is too busy moving data in and out of memory rather than actually executing the program.

程序执行停止的点，因为系统忙于在内存中移入和移出数据而不是实际执行程序。
