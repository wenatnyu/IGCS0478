ROM - Read-Only Memory

只读存储器

Key Features / 主要特征

  • Non-volatile memory / 非易失性存储器

    Contents retained when power off / 断电后内容保留

  • Permanent memory / 永久存储器

    Cannot be changed by user / 用户不能更改

    Data can only be read / 数据只能读取

  • Contains essential startup data / 包含启动所需数据

    BIOS (Basic Input/Output System) / 基本输入输出系统

    Bootstrap instructions / 引导程序

    Factory settings / 出厂设置

Common Uses / 常见用途

  • Computer startup routines / 计算机启动程序

  • Embedded systems / 嵌入式系统

  • Firmware storage / 固件存储

  • Remote control devices / 遥控设备
