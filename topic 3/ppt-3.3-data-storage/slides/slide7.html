Technology Overview / 技术概述

  • Magnetic storage device

    磁性存储设备

  • Mechanical device with moving parts

    有移动部件的机械设备

  • Data stored digitally on magnetic surfaces

    数据以数字形式存储在磁性表面

Key Components / 主要组件

  • Platters / 磁盘盘片

    Magnetic disks (aluminum, glass, or ceramic) that spin at ~7000 RPM. Multiple platters in one drive, each has two usable surfaces. / 磁盘(由铝、玻璃或陶瓷制成), 以约7000转/分钟旋转。一个驱动器中有多个盘片, 每个盘片有两个可用表面。

  • Read-Write Heads / 读写磁头

    Electromagnets that read/write data by accessing all platter surfaces. They move very quickly: 50 times per second from center to edge. / 读写数据的电磁体, 访问所有盘片表面。移动速度快：每秒从中心到边缘50次。

  • Spindle Motor / 主轴电机

    Rotates platters at high speed. / 高速旋转盘片。

  • Actuator Arm / 驱动臂

    Positions read-write heads over the correct track. / 定位读写磁头。

Data Organization / 数据组织

  • Tracks / 磁道

    Concentric circles on disk surface. / 磁盘表面的同心圆。

  • Sectors / 扇区

    Fixed-size blocks on tracks, each holding a set number of bytes. Data is accessed directly by sector. / 磁道上的固定大小块, 每个扇区包含固定字节数。可直接访问特定扇区。

Performance Issues / 性能问题

  • Latency / 延迟

    Time for a data block to rotate to the read-write head. Causes 'Please wait' messages. / 数据块旋转到读写磁头的时间, 导致"请稍候"消息。

  • Fragmentation / 碎片化

    A file's sectors may not be adjacent due to edits and deletions, degrading performance over time. / 文件扇区可能不相邻, 长期使用导致性能下降。

Characteristics / 特性

  • High capacity / 大容量
  • Lower cost per GB / 每GB成本低
  • Slower than SSD / 比SSD慢
  • Removable versions available for backup / 可移动版本用于备份
